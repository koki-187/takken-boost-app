# 宅建BOOST 取扱説明書
バージョン: v5.2.0  
最終更新: 2025年9月28日

## 📱 はじめに

宅建BOOSTは、宅地建物取引士試験の合格を目指すあなたのための学習アプリです。
スマートフォン、タブレット、パソコンなど、あらゆる端末でご利用いただけます。

## 🚀 インストール方法

### iPhone/iPadをお使いの方

1. **Safari**を開きます（必ずSafariを使用してください）
2. アドレスバーに以下のURLを入力します：
   ```
   https://master.takken-boost.pages.dev
   ```
3. ページが表示されたら、画面下部の「**共有**」ボタンをタップ
   ![共有ボタン](□に↑マーク)
4. メニューから「**ホーム画面に追加**」を選択
5. アプリ名が「宅建BOOST」になっていることを確認
6. 右上の「**追加**」をタップ
7. ホーム画面にアイコンが追加されます

### Androidをお使いの方

1. **Chrome**を開きます
2. アドレスバーに以下のURLを入力します：
   ```
   https://master.takken-boost.pages.dev
   ```
3. ページが表示されたら、画面上部または下部に表示される
   「**ホーム画面に追加**」バナーをタップ
   - バナーが表示されない場合：
     1. 右上の「⋮」メニューをタップ
     2. 「**アプリをインストール**」または「**ホーム画面に追加**」を選択
4. インストールダイアログで「**インストール**」をタップ
5. ホーム画面にアイコンが追加されます

### Windows/Macをお使いの方

#### Chrome/Edgeの場合：
1. ブラウザでアプリを開きます：
   ```
   https://master.takken-boost.pages.dev
   ```
2. アドレスバーの右端にある「**インストール**」アイコン（⊕マーク）をクリック
3. 「**インストール**」ボタンをクリック
4. デスクトップまたはアプリ一覧に追加されます

## 📖 使い方ガイド

### 初回起動時

アプリを初めて開くと、チュートリアルが自動的に始まります。
- 「**次へ**」ボタンで進む
- 「**スキップ**」で省略可能

### メイン画面の見方

#### 📊 上部の統計カード

1. **学習進捗**（青色）
   - 402問中、何問完了したかを表示
   - パーセンテージで進捗を確認

2. **正答率**（緑色）
   - これまでの正解率を表示
   - 学習の精度を確認

3. **連続学習**（オレンジ色）
   - 毎日の学習継続日数
   - モチベーション維持に活用

### 📚 学習を始める

#### カテゴリー学習

1. メイン画面の「**カテゴリー学習**」ボタンをタップ
2. 4つのカテゴリーから選択：
   - **権利関係**（紫）: 民法、借地借家法など
   - **法令上の制限**（青）: 都市計画法、建築基準法など
   - **税・その他**（緑）: 税法、価格評定など
   - **宅建業法**（オレンジ）: 宅建業法、住宅瑕疵担保法など
3. 問題が表示されます
4. 4つの選択肢から正解を選びます
5. 回答後、自動的に解説が表示されます
6. 「**次の問題へ**」で続けます

#### 問題の見方

- **問題番号**: 左上に「問題 1 / 10」のように表示
- **カテゴリー名**: 現在学習中の分野
- **選択肢**: A〜Dの4つから選択
- **解説**: 回答後に表示される詳しい解説

### 📝 模擬試験

1. メイン画面の「**模擬試験**」ボタンをタップ
2. 試験概要を確認：
   - 問題数: 50問
   - 制限時間: 2時間
   - 合格ライン: 35問以上（70%）
3. 「**試験を開始する**」で本番開始

> ⚠️ **注意**: 現在、模擬試験機能は開発中です

### 📈 学習結果の確認

学習セッション終了後、結果画面が表示されます：
- **正答率**: 大きな数字で表示
- **正解数/問題数**: 詳細な成績
- **正解・不正解の内訳**: 緑と赤のカードで表示

### 🔧 その他の機能

#### ヘルプボタン
右上の「❓」アイコンをタップすると、いつでもチュートリアルを確認できます。

#### ナビゲーションバー（画面下部）
- **ホーム**: メイン画面へ
- **学習**: カテゴリー選択へ  
- **統計**: 学習統計（開発中）
- **プロフィール**: ユーザー設定（開発中）

## ❓ よくある質問

### Q: オフラインで使えますか？
A: 一度アクセスした内容は、オフラインでも表示可能です。ただし、初回は必ずインターネット接続が必要です。

### Q: 学習データは保存されますか？
A: はい、お使いの端末に自動的に保存されます。アプリを閉じても、次回起動時に継続できます。

### Q: 他の端末と同期できますか？
A: 現在、端末間の同期機能は開発中です。

### Q: 問題数が少ないのですが？
A: 現在サンプル問題のみ実装されています。402問の完全版は順次追加予定です。

## 🔧 トラブルシューティング

### アプリがインストールできない
- **iOS**: Safariを使用しているか確認してください
- **Android**: Chromeを最新版に更新してください
- ブラウザのキャッシュをクリアしてから再試行してください

### アイコンが正しく表示されない
- 一度アンインストールして、再インストールをお試しください

### 学習データが消えた
- ブラウザのデータ削除を行うと、学習データも削除されます
- 定期的にスクリーンショットで記録を残すことをお勧めします
